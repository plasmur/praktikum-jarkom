## tujuan setiap command 
1. bertanya suatu server dns dan mendapatkan informasi dns tenetang server tersebut (nslookup www.mit.edu)
2.  mengirimkan permintaan untuk record tipe-NS ke default server DNS lokal (nslookup -type=NS mit.edu)
3. mengirim kueri untuk server dns tertentu bukan untuk dns default (nslookup www.aiit.or.kr bitsy.mit.edu)
4. menampilkan informasi mengenai tcp/ip dll (ipconfig/all) alternative (ipconfig /all > networkinfo.txt) kalau alternative 
5. menampilkan change dns yang tampil di pc kita (ipconfig /displaydns)
6. manghapus semua chache dari dns (ipconfig /flashdns)
